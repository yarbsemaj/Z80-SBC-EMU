Monitor commands

When the emulator first starts it will be running the monitor program, this gives a window into the machine that can be used to load other programs. When in the monitor program the following commands can be used.
R nn Dumps out 0xFF bites of memory starting at address 0xnn00.
W nnnn xx Write byte xx to address 0xnnnn.
E nnnn Runs a program starting at address 0xnnnn.

Start Basic

Basic is in memory at address 0x0100, typing E 0100 in the monitor will start it.
Hit enter when you see Memory Top?

Load Roms

ROM images can be loaded using the on-screen menu. All ROM are loaded into memory starting at address 0x9000 and start immediately.
All action games use WASD to control.

Load iHEX files

iHEX Files can be loaded using the on-screen menu and executed using the monitor.
